# breadboi-tv

My personal website for all things streaming and social. You may view the live site [here](https://breadboi.tv/)

## Requirements

- NPM
- Go
- Gulp
- config.json ( For config.go )

## config.json
```
{
  "port" : <port>,
  "mode" : <debug/release>
}
```

## Startup

```bash
> npm install
> gulp babel
> go run *.go
```
